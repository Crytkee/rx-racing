What is RX Racing?
RX Racing is an incredible game made possible by It's a multiplayer game where you try to complete three laps before the other players.
How do I play?
First, type in a name and use the color slider to choose the color of your car. If you want to host a game, or if you want to play alone, click
Host a Game
If someone else is hosting a game, press
Join a Game
When hosting, have all other players type in your four-letter code to join. If someone else is hosting, type in the code on their screen.

If you're on a computer, use the left and right arrow keys to steer. Your car will automatically drive forwards. On a phone, with or without VR mode, tilt your phone to steer.

Be careful not to hit other players!
